# Cookaroo
An application to view and edit recipes.

Instructions:
1. Download the source code.
2. Run using: 'ng serve'
3. Download the source code of the backend at the following url: https://github.com/ilanamost/cookaroo-backend
4. Run the backend using: 'npm start'.
5. Open the browser to http://localhost:4200

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.1.6.
